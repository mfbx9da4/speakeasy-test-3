# ShortURLs
(*shortURLs*)

## Overview

REST APIs for managing short URLs

### Available Operations

* [create](#create) - Shorten a URL.

## create

Shorten a URL.

### Example Usage

```typescript
import { SDK } from "petstore";

const sdk = new SDK({
  security: {
    apiKey: "<YOUR_API_KEY_HERE>",
  },
});

async function run() {
  const result = await sdk.shortURLs.create({
    url: "https://probable-heating.com/",
  });

  // Handle the result
  console.log(result);
}

run();
```
